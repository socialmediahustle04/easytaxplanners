## How To Use These Prompts In ChatGPT

1. Generate one image at a time.
2. Use the exact filename as the label for each image.
3. Download each final image as PNG.
4. Save all images into `/Users/apple/easytaxplanners/public/images/generated/`.
5. Review all images together before any website integration.
6. Do not integrate into code until every image is approved for originality, brand fit, cropping, and no sensitive-data issues.
