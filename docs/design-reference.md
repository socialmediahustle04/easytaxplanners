# Safe Design Reference

## Important

This is not a ClearTax asset folder.

Do not copy ClearTax files, images, CSS, illustrations, screenshots, exact layouts, logos, or proprietary brand elements.

This document captures safe UX lessons from public tax platforms and converts them into an original EasyTaxPlanners design direction.

## Useful Competitor UX Patterns

Large tax platforms usually do a few things well:
- Clear top navigation
- Product/service grouping
- Strong first CTA
- Service cards
- Trust badges
- Pricing/service segmentation
- Dense footer with SEO links
- Repeated reassurance throughout the page

EasyTaxPlanners should use these general product principles, not their exact designs.

## Original EasyTaxPlanners Design Direction

### Personality

Modern CA office meets compliance dashboard.

The site should feel:
- Clear
- Calm
- Professional
- Trustworthy
- Useful from the first screen

### Visual System

Recommended palette:
- Background: `#F7FAFC`
- Surface: `#FFFFFF`
- Primary text: `#17233A`
- Muted text: `#65738A`
- Primary CTA: `#2563EB`
- Deep trust navy: `#0B1F3A`
- Success green: `#16A34A`
- Warning amber: `#F59E0B`
- Border: `#D7E1F0`

Use the palette flexibly. Do not make the UI look like a one-color blue template.

### Typography

Use a clean modern sans-serif:
- Inter
- Geist
- Manrope
- Plus Jakarta Sans

Recommended:
- Headings: bold, tight but readable
- Body: 16-18px
- Service cards: compact and scan-friendly

### Hero Visual

Create an original dashboard-style mockup showing:
- Service request received
- Document checklist
- Filing status
- Pending review
- Completed filing

Do not use generic smiling office photos as the main hero.

### Homepage Layout Rhythm

1. Hero with CTA and dashboard visual
2. Six core service cards
3. "Who we help" section
4. "How it works" process
5. Trust / CA-led guidance section
6. Problem-led section
7. Resources preview
8. Final CTA

### Service Cards

Each service card should show:
- Icon
- Service name
- One-line explanation
- 3 short bullets
- CTA: "Get help"

Cards:
- Income Tax Filing
- GST Return Filing
- Company Return Filing
- GST Registration
- Company Incorporation
- Financial Statements

### Motion

Use subtle motion only:
- Fade-in sections
- Hover lift on service cards
- Soft checklist/status transitions

Avoid:
- Heavy animated backgrounds
- Decorative gradient blobs
- Overly playful motion

## Trust Components

Build components for:
- CA-led guidance badge
- Document checklist badge
- WhatsApp support badge
- Draft/review before filing badge
- Transparent scope badge
